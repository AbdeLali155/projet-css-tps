1
Sélecteurs et Spécificité
Cibler des éléments HTML avec précision

1.1 — Sélecteurs CSS

Solution

css
/* 1. Tous les liens dans la navigation */
.main-nav a { }

/* 2. Uniquement le lien actif */
.main-nav a.active { }

/* 3. L'article avec la classe "featured" */
article.post.featured { }

/* 4. Le premier paragraphe de chaque article */
article.post p:first-of-type { }

/* 5. Les éléments li pairs */
li:nth-child(even) { }

1.2 — Spécificité

Ordre croissant de spécificité

SélecteurScore (a,b,c)Niveau
p(0,0,1)Très faible
.text(0,1,0)Faible
p.text(0,1,1)Moyen-bas
#main .text p(1,1,1)Élevé
#main(1,0,0)Élevé

Même spécificité → la règle déclarée en dernier dans le CSS gagne (principe de la cascade).

Éviter !important → il brise la cascade naturelle, rend le débogage très difficile et crée des conflits en chaîne.

2
Box Model
Comprendre le calcul des dimensions

2.1 — Calcul des largeurs

Réponses

content-box (défaut) : 300 + 20×2 + 5×2 = 350px de largeur totale visible

border-box : La largeur totale reste 300px — padding et border sont inclus dans le width

css
/* Reset universel box-sizing */
*, *::before, *::after {
  box-sizing: border-box;
}

2.2 — Centrage horizontal + vertical

Solution

css
/* Centrage horizontal seulement */
.centered {
  width: 600px;
  margin: 0 auto;
}

/* Centrage horizontal ET vertical avec Flexbox */
body {
  min-height: 100vh;
  display: flex;
  align-items: center;
  justify-content: center;
}

.centered {
  width: 600px;
}

3.2 — Cartes de même hauteur

Solution — footer toujours en bas

css
.cards {
  display: flex;
  gap: 24px;
  align-items: stretch; /* hauteur égale pour toutes les cartes */
}

.card {
  display: flex;
  flex-direction: column;
  flex: 1;
  padding: 24px;
  border: 1px solid #ddd;
  border-radius: 8px;
}

.card p {
  flex: 1; /* pousse le bouton vers le bas */
}

.card-link {
  margin-top: auto;
  display: inline-block;
  padding: 8px 16px;
  background: #0077ff;
  color: #fff;
  text-decoration: none;
  border-radius: 6px;

4
CSS Grid
Grilles et mise en page avancée

4.1 — Grille responsive automatique

Solution

css
.grid-cards {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 24px;
}

.card {
  padding: 20px;
  border: 1px solid #ddd;
  border-radius: 8px;
}

/* auto-fit + minmax → adapte le nombre de colonnes
   automatiquement : 3 sur desktop, 2 sur tablette, 1 sur mobile */

4.2 — Layout complet avec Grid Areas

Solution

css
.page-layout {
  display: grid;
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
  grid-template-columns: 250px 1fr;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
  gap: 16px;
}

.header  { grid-area: header;  }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main;    }
.footer  { grid-area: footer;  }
5
Responsive Design
Approche Mobile First avec media queries

Solution — Mobile First

css
/* ── Base mobile (styles par défaut) ── */
.container { width: 100%; padding: 15px; }

.nav-menu {
  display: flex;
  flex-direction: column;
  gap: 5px;
}

.content { display: block; }
.sidebar { display: none; }

/* ── Tablette : ≥ 768px ── */
@media (min-width: 768px) {
  .container {
    max-width: 768px;
    margin: 0 auto;
    padding: 20px;
  }

  .nav-menu {
    flex-direction: row;
    gap: 16px;
  }

  .content {
    display: grid;
    grid-template-columns: 1fr;
    gap: 16px;
  }

  .sidebar { display: block; }
}

/* ── Desktop : ≥ 1024px ── */
@media (min-width: 1024px) {
  .container { max-width: 1200px; }

  .content {
    grid-template-columns: 1fr 300px;
  }
}

Variables CSS et Thèmes
Système de design avec custom properties

Solution — Thème clair / sombre

css
/* ── Thème clair (défaut) ── */
:root {
  --color-primary:    #0077ff;
  --color-secondary:  #6c757d;
  --color-bg:         #ffffff;
  --color-surface:    #f8f9fa;
  --color-text:       #212529;
  --color-text-muted: #6c757d;
  --color-border:     #dee2e6;
  --color-success:    #28a745;
  --color-danger:     #dc3545;
  --border-radius:    8px;
  --shadow:           0 2px 8px rgba(0,0,0,0.1);
}

/* ── Thème sombre ── */
[data-theme="dark"] {
  --color-primary:    #4da6ff;
  --color-bg:         #1a1a2e;
  --color-surface:    #16213e;
  --color-text:       #e9ecef;
  --color-text-muted: #adb5bd;
  --color-border:     #495057;
  --shadow:           0 2px 8px rgba(0,0,0,0.4);
}

/* ── Utilisation ── */
body {
  background-color: var(--color-bg);
  color: var(--color-text);
}

.card {
  background: var(--color-surface);
  border: 1px solid var(--color-border);
  border-radius: var(--border-radius);
  box-shadow: var(--shadow);
}

/* Activation via JavaScript */
/* document.documentElement.setAttribute('data-theme', 'dark'); */

Transitions et Animations
Bouton animé et spinner de chargement

7.1 — Bouton animé

Solution

css
.btn {
  padding: 12px 28px;
  background: #0077ff;
  color: #fff;
  border: none;
  border-radius: 6px;
  cursor: pointer;
  font-size: 16px;
  transition: background 0.3s ease,
              transform  0.2s ease,
              box-shadow 0.3s ease;
}

.btn:hover {
  background: #005fcc;
  transform: translateY(-2px);
  box-shadow: 0 6px 16px rgba(0, 119, 255, 0.35);
}

.btn:active {
  transform: translateY(0);
  box-shadow: none;
}

7.2 — Spinner de chargement

Solution

css
@keyframes spin {
  from { transform: rotate(0deg);   }
  to   { transform: rotate(360deg); }
}

.spinner {
  width: 40px;
  height: 40px;
  border: 4px solid #e0e0e0;
  border-top-color: #0077ff;
  border-radius: 50%;
  animation: spin 0.8s linear infinite;
}
